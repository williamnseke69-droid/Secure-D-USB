# Secure-D-USB
Secure D. USB est un scanner de sécurité USB en temps réel qui détecte les menaces avant qu'elles n'atteignent votre système. Analyse multi-niveaux (signatures, entropie, métadonnées, VirusTotal), scoring intelligent, et base de signatures communautaire partagée entre utilisateurs. Verdict en moins de 10 secondes, ne ralentit pas votre travail
